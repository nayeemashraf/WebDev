Flex is a display property which allows us to alter the elements within the parent container to make them responsive. We can adjust the direction, flow, alignment, size, order, etc of the flex items using this property. The direct children of the flex container are the flex items upon which all the changes are applied.

If our content flows horizontally, in the row direction, that will be our main axis. However, if our content flows vertically, in a column direction, that will be our main axis, and the axis perpendicular to the main axis is called the cross axis.
